# Student Performance Predictor

This project uses Machine Learning to predict student marks based on study hours.

## Technologies Used
- Python
- scikit-learn
- pandas

## Features
- Train model
- Predict marks
- Data visualization
## 🌐 Web Application

This project has been extended into an interactive web application using Streamlit.

### Features:

* User-friendly interface to input study hours
* Real-time prediction of student marks
* Machine Learning model integration (Linear Regression)
* Data visualization support

### How to Run:

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

3. Open in browser:
   http://localhost:8501
